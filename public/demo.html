Fitur Unggulan

Semua yang Anda Butuhkan

Dari capture baseline hingga laporan HTML — semua dalam satu alur.

📸

Screenshot Otomatis

Capture baseline langsung dari URL tanpa perlu upload manual. Full-page scroll otomatis untuk konten lazy-load.

🔁

Journey Multi-Checkpoint

Test UI sebelum login, sesudah login, dan halaman dalam satu sesi browser. State tersimpan antar checkpoint.

🎯

Pixel-Level Comparison

Powered by pixelmatch. Deteksi perbedaan sekecil 0.001% dengan laporan diff image berwarna merah.

🙈

Sembunyikan Elemen Dinamis

CSS selector atau XPath untuk menyembunyikan jam, iklan, atau konten berubah sebelum screenshot diambil.

✅

Content Assertion

Verifikasi teks pada elemen tertentu. Pastikan tombol bertuliskan "Login" dan bukan "Sign In".

📦

Export / Import Konfigurasi

Simpan seluruh journey termasuk baseline ke file JSON. Import kembali kapan saja — tidak perlu input ulang.
